API / FILE

This endpoint is for sending SHA256 & SHA512 file hashes to determine if a file is known to be malicious.

At no point should this endpoint be used in a production environment as the SOLE antimalware solution! This endpoint is not guaranteed to be 100% accurate.

File Object

Field Data Type Description
sha256 String The sha256 hash of the file
sha512 String The sha512 hash of the file
classification? String The classification of the file (if known and malicious)
fileType? String The file/mime type
expectedExtension? String The expected file extension
detectionDate? u64 The estimated date the file was detected (epoch millisecond)
additionDate u64 The date the file was added to the local database (epoch millisecond)
scanDate u64 The date the file itself was last checked (epoch millisecond)

API / LOGGING

This endpoint is a private endpoint for observing and managing the various logs from all astraDev managed services.

This endpoint is secured and requires authentication and authorization for usage.

Logging Object

Field Data Type Description
id u64 The log id
service String The service the log oringated from
level String The log severity level
message String The log message
time u64 The date the log was added to the local database (epoch millisecond)
trace? String The stacktrace
className String The class the log came from

Routes

GET
/logging
Returns an estimated count of logs per service

GET
/logging?id=u64
Returns an Log object from the log ID

GET
/logging?page=u32
Returns a list of Log objects for a page (10 logs per page)

GET
/logging?service=STRING
Returns a list of Log objects based on the service (10 per page if coupled with page)

GET
/logging?critical=BOOLEAN
Returns a liast of Log objects based on the high critical levels (10 per page if coupled with page)

PATCH
/logging
Returns 200 on successful log escalation or 204 on failure to escalate

JSON

Key Data Type
id u64

DELETE
/logging
Returns 200 on successful log deletion or 204 on failure to delete

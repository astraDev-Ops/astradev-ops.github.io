API / FILES

This endpoint is for sending domains to determine if a link is known to be malicious.

At no point should this endpoint be used in a production environment as the SOLE antimalware solution! This endpoint is not guaranteed to be 100% accurate.

Link Object

Field Data Type Description
domain String The website domain + subdomains
isWhitelisted boolean If the domain is internally whitelisted
banned? String The reason the domain is banned

Routes

GET
/link?domain=domain
Returns a Link object

FILE


The file endpoint is for the potential detection of malicious files by hash. File uploading is not supported at this time.

GET /file?query={ sha256 }
THIS ENDPOINT DOES NOT REPLACE THE USE OF COMMON SENSE AND YOUR OWN LOCALLY INSTALLED ANTI-MALWARE SOLUTION
This endpoint supports an array of up to 5 hashes per request.

Check a file's SHA256 hash with our database to see if a file MIGHT be malicious.

Returns a FILE object.


GET /file?page={ page_number }
Accounts accessing this endpoint require the following permissions: ARRAY_ACCESS

Get a certain page of data (returns max 50 results per page)

Returns an array of FILE objects.


PATCH /file
Accounts accessing this endpoint require the following permissions: MANAGE_FILE

Modify the File data

JSON Parameters
Field Data Type Notes
sha256 String The file's sha256 hex as string
classification String The OPTIONAL classification as String
sample_available Boolean The OPTIONAL sample available result as boolean

Returns Status: 200

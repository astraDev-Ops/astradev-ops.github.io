Quiz


The quiz endpoint is for generating random quizes.

GET /quiz?amount={ INTEGER }
This endpoint will only return a maximum of 10 items.
You can also specify which category by adding &category={ STRING } to the request.

Returns an array of Quiz objects.


GET /quiz/review
Accounts accessing this endpoint require the following permissions: MANAGE_QUIZ

This endpoint will return a single pending quiz awaiting validation.

Returns a pendingQuiz object.


POST /quiz

Create a new pending quiz.

JSON Parameters
Field Data Type Notes
id Long The id within the database
question String
answer String
wrong_a String The first wrong answer
wrong_b String The second wrong answer
wrong_c String The third wrong answer
category String The category of the quiz
tags String This field is depreciated and will be removed soon

Returns Status: 200


PATCH /quiz
Accounts accessing this endpoint require the following permissions: MANAGE_QUIZ

Approve a quiz for general usage

JSON Parameters
Field Data Type Notes
id long A non-null long of the quiz id

Returns Status: 200


DELETE /quiz
Accounts accessing this endpoint require the following permissions: MANAGE_QUIZ
JSON Parameters
Field Data Type Notes
id long A non-null long of the quiz ID

Returns Status: 200

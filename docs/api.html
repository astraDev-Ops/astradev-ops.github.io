Basic Setup

All requests to the API server are to be done with HTTP requests. All endpoints require authentication, some require authorisation.

Creating an account
In order to access the API endpoints please ensure you have a valid API key from here.
Headers
Header Name Required Value Notes
x-auth-key YOUR API KEY This is a required header. Please ensure you are sending this in all requests.
x-audit-reason A 10+ CHAR STRING This is a required header for some endpoints. Endpoints requiring this header will be marked accordingly.

Permissions

The bot requires certain permissions to function properly, if you do not want these permissions to be granted to the bot that is ok

Permission Function
View Channels Allows the bot to see channels
View Audit Log For logging functions
Manage Webhooks For logging functions
View Audit Log For logging functions
Kick Members To kick users
Ban Members To ban users
Timeout Members To timeout users
Send Messages For sending messages that are not interactions
Send Messages in Threads For sending messages that are not interactions in thread channels
Use External Emoji For sending emojis within the app's directory of emojis
Manage Messages For automoderation features
Manage Threads For monitorring activities in private threads (automod)
Read Message History For automoderation features
